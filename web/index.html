平台说明

  • Linux / Windows：推荐 Chrome 或 Edge，选择“分享标签页/窗口/屏幕”时勾选音频共享。
  • Android：多数浏览器不开放系统音频采集；默认使用麦克风模式。若部署为 HTTPS/PWA，可直接在手机浏览器访问。
  • 安全上下文：音频采集通常要求 HTTPS 或 localhost。请不要直接双击 file:// 方式长期使用。
